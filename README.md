# Guia-Digital_Comunitario

Projeto feito para a Univeridade Algusto Motta (Unisuam) - Bangu

Este projeto tem como finalidade ser um guia turistico para as pessoas, sobre as 7 maravilhas do mundo moderno:

Grande Muralha da China (China),
Cristo Redentor (Brasil),
Machu Picchu (Peru),
Chichén Itzá (México),
Coliseu (Itália),
Petra (Jordânia),
Taj Mahal (Índia)

Como forma de site, eu decidi fazelo para incentivar mais as viajem ao exterior.
